# 🐍 Snake Water Gun Game (Python)

A simple command-line Snake Water Gun game built using Python.

This game is based on the classic game where the player competes against the computer.

## 🎮 Game Rules

- 🐍 Snake beats 💧 Water
- 💧 Water beats 🔫 Gun
- 🔫 Gun beats 🐍 Snake


## 🚀 Features

- Player vs Computer
- Random computer choice
- Simple command-line gameplay
- Beginner-friendly Python project
- Uses dictionaries and conditional statements


## 🛠️ Technologies Used

- Python 3
- Random module
